Actualización (0.8 beta):

 

 - Mejorada la jugabilidad:

 Ahora tiene una jugabilidad mucho mejor a la anterior, los daños de ataques están cambiados a un nivel más realista a la pelea (no exagerados).

Y más cosas que descubrirán al jugarlo.

 - A.I. Mejorada:

 La A.I. tiene el nivel que usa mugen, hasta 8 niveles diferentes.

 - Cambiados y mejorados los supers y especiales.

 Supers con efectos de SuperChargue y combinaciones entre ellas.

 Algunas especiales fueron cambiadas por supers.

 Kamehame-Ha tiene un modo de carga para hacer mas daño

 Superball cancelado, tal vez después lo mejore.

 - Agregado un Super “Rolling Throw”.

 - Fireball y Zanzoken ahora con límites de repeticiones.

 Fireball con máximo de 6 repeticiones y Zanzoken con un acumulador de uso.

 - Mejorado el Modo Cansado:

 Ahora tiene un indicador que muestra el nivel de cansancio.

 Bugs arreglados.

 Y otras cosas mas mejoradas en este modo.

 - Compatibilidades de animaciones con otros chars:

Electrocutado, congelado, quemado y algunas compatibles con chars (MK) de Juano .

- Más presentaciones y poses de victoria.

- Más y más cosas que notaran si los prueban

 Leyenda para Tipos De Estados:

 

 A = Aéreo

 S = Suelo

 V = Volando

 

 Con el poder mínimo para hacer el movimiento, puedes hacer el movimiento luego de esto, el poder ira disminuyendo, si queda muy poco poder el personaje quedara cansado. Aparecerá una barra indicadora que mostrará el cansancio acomunado.

 El tiempo de cansancio aumenta cuando el char esta haciendo el movimiento. El personaje seguirá cansado hasta que se cumpla hasta que se cumpla el tiempo establecido, esto no será cancelado por golpes del rival. Disminuye este tiempo presionando cualquier botón. Si el personaje esta descansado (animación) será más rápido la descarga.

 Este modo puede ser muy beneficioso cuando el personaje no tenga suficiente poder para realizar algún ataque, pero muy incomodo si el cansancio es elevado.

 

Recomendaciones:

 

 En el archivo "EGC-kid_Goku_GT.cns" se puede cambiar el número de estas líneas:

 

 [Data]

 Life = 1000

 Vida máxima de este personaje.

 Lo normal es 1000

 Attack = 100                         

 Fuerza de ataque de este personaje.

 Lo normal es 100

 Defense = 100

 Defensa de este personaje.

 Lo normal es 100

 Power = 300

 Poder máximo de este personaje.

 Lo normal es 3000

 No es recomendable cambiar este valor. Si pones un número mas bajo puede que no se pueda ejecutar algunos ataques. Estoy trabajando para que se pueda cambiar al gusto que quieran.

 

 Si quieren cambiar el tamaño de Goku, abran el archivo "EGC-Kid_Goku_GT.def" con un editor de texto (Ej. Notepad) y lean la explicación que se encuentra dentro de este.

 

 La característica principalmente de este personaje es de hacer combos, así que presiona los botones de ataques para interrumpir un ataque y pasar a otro. Algunos combos requieren más de un botón para ejecutarse. Ej.: F+A
